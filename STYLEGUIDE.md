Style Guide - Social Media Content Analyzer

Palette
- Primary background: #0b0b0b (deep black)
- Card surface: #0f0f0f (dark)
- Accent gradient: #ff6a00 â†’ #ff9400 (vibrant orange)
- Muted text: #a1a1a1
- Danger: #ff5a5a

Typography
- Font family: Inter, system-ui fallbacks
- Headings: 700 weight, accent color for main title
- Body: 400 weight, light color on dark background

Components
- Panels: dark gradient surface, subtle border, rounded corners (12px)
- Buttons: bold orange gradient, rounded 10px, primary action
- Secondary buttons: transparent with subtle border
- Dropzone: dashed border, dark gradient background, highlight with orange on drag

Spacing
- Comfortable padding: 16-20px inside panels
- Gap between main content and sidebar: 20px

Accessibility
- Text contrast: ensure body and controls maintain 4.5:1 contrast where possible
- Keyboard focus: buttons should have visible outline (browser default is acceptable)

Usage
- Use server OCR for scanned PDFs; client-side OCR for small images.

Notes
- For production, consider replacing Google Fonts link with an offline font or self-hosted font for privacy.
